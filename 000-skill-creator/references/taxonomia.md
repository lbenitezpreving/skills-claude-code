# Taxonomia de Codigos de Skills

```
0xx  Documentacion y especificacion
     000  Creador de skills (esta skill)
     001  Generador de especificaciones de proyecto
     002  Desglose de especificaciones en tareas
     003  Generacion de ADR (Architecture Decision Records)

1xx  Frontend
     101  Componente React/TypeScript
     102  Componente Angular
     103  Pagina completa (React Router / lazy loading)
     104  Hook personalizado React

2xx  Backend
     201  Endpoint FastAPI
     202  Endpoint Express/Node
     203  Servicio de dominio (logica de negocio)
     204  Middleware / interceptor

3xx  Base de datos
     301  Modelo ORM + migracion (SQLAlchemy)
     302  Seed / datos iniciales
     303  Query compleja / reporte SQL

4xx  Revision de codigo
     401  Revision general de cambios recientes
     402  Auditoria de seguridad
     403  Revision de tests / cobertura

5xx  Swagger / Postman / Contratos API
     501  Coleccion Postman desde rutas FastAPI
     502  Exportar OpenAPI / Swagger
     503  Contrato API (consumer-driven)

6xx  Gestion con Git
     601  Commit inteligente con co-autoria
     602  Gestion de ramas y pull requests
     603  Changelog automatico

7xx  Despliegues
     701  Dockerfile y docker-compose
     702  Pipeline CI/CD (GitHub Actions)
     703  Despliegue en cloud (configuracion)
```
